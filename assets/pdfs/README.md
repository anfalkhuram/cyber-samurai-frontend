# Resume PDF

This directory should contain:

- resume.pdf - A PDF version of the resume

You can create this PDF by:
1. Opening resume.html in a browser
2. Using the browser's print function
3. Saving as PDF
4. Or using a tool like wkhtmltopdf or similar

The PDF should be optimized for printing and sharing.
